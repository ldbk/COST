\HeaderA{PerformStats}{Method for computing performance statistics}{PerformStats}
\aliasA{PerformStats-methods}{PerformStats}{PerformStats.Rdash.methods}
\keyword{methods}{PerformStats}
\begin{Description}\relax
The function computes the performance statistics to evaluate accuracy, bias and precision of different estimators. The outcome 
object is of the performStats class.
\end{Description}
\begin{Usage}
\begin{verbatim}
performStats(estSimObj, trueDataObj, desc, nSamples)
\end{verbatim}
\end{Usage}
\begin{Arguments}
\begin{ldescription}
\item[\code{estSimObj}] A dbeOutputSim object
\item[\code{trueDataObj}] A trueData object
\item[\code{desc}] Object description
\item[\code{nSamples}] Not needed anymore?
\end{ldescription}
\end{Arguments}
\begin{Value}
An object of the class performStats
\end{Value}
