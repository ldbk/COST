\HeaderA{dbeOutputSim}{Class "dbeOutputSim"}{dbeOutputSim}
\keyword{classes}{dbeOutputSim}
\begin{Description}\relax
Outcome object from applying \code{COSTdbe} methods to simData class objects. It has the same main 
estructure as dbeOutput class objects but the dataframes generated by the estimation methods include a
variable \code{sample} which identifies the simulated data set.
\end{Description}
\begin{Section}{Objects from the Class}
The creator function \code{dbeOutputSim} can be called to create objects from this class.
\end{Section}
